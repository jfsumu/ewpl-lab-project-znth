# ZnthHost - Web Hosting Business Landing Page

A modern, responsive, and interactive web hosting business landing page built using **HTML5**, **CSS3**, and **JavaScript (ES6)**. The project is designed to simulate the front-end experience of a professional hosting provider, allowing visitors to explore hosting services, compare pricing plans, and experience a realistic purchasing workflow.

## 📖 Project Overview

ZnthHost is a front-end web application that showcases the services of a fictional web hosting company. The website features a clean, modern interface with responsive layouts, smooth animations, pricing cards, interactive FAQs, and a simulated checkout process to provide users with an engaging browsing experience.

This project was developed as an educational demonstration of modern web design principles and front-end development.

Frequently Asked Questions

Common Queries

Get answers to the most common questions regarding our server offerings, architecture details, and administrative policies.

What is VPS Hosting?

VPS stands for Virtual Private Server. VPS hosting works by partitioning a single physical host server into multiple virtual environments using hypervisor technology. Unlike standard shared web hosting, you get fully dedicated allocations of memory (RAM), storage (SSD), and CPU cycles that are isolated from other accounts.

What is the difference between Managed and Unmanaged VPS?

In Managed VPS hosting, our technical team handles setup operations, regular updates, patches, firewall parameters, security monitoring, and backup restores. Unmanaged VPS hosting gives you root level OS shell commands to configure your servers manually. It is best suited for experienced developers who want direct container control and do not require administrative assistance.

Do you provide backups?

Yes, our hosting systems include off-site data vaults. Business Hosting, Managed VPS, and Dedicated servers feature automated daily system snapshots with 7-day version histories. Shared and Unmanaged plans include manual backup utilities within the hosting cPanels.

Is technical support available 24/7?

Absolutely. ZnthHost provides 24/7/365 customer support. Our server experts can be contacted anytime via live chat or our ticket portal to troubleshoot infrastructure bottlenecks, routing issues, or hardware anomalies.

Can I upgrade my hosting plan later?

Yes, upgrading can be triggered instantly from the customer portal. Our virtualization layer allows scaling CPU cores, memory limits, and disk space seamlessly. The migration process requires zero downtime and carries over all existing system files and configurations automatically.

Do you offer SSL certificates?

Yes, free Let's Encrypt Wildcard SSL certificates are generated automatically for all domain names and subdomain configurations hosted on ZnthHost servers, providing HTTPS encryption out of the box.

What is your uptime guarantee?

ZnthHost delivers a 99.9% service level agreement (SLA) uptime guarantee. Our data centers feature redundant power supplies, multiple telecom uplink nodes, and active hardware clustering to keep websites online continuously.

How long does server deployment take?

Standard VPS plans and shared hosting environments are provisioned automatically within 60 seconds after payment confirmation. Dedicated Bare-metal server setups require human checks and are ready for terminal commands within 1 to 4 hours.
